### 접근
1. 큐를 이용해 풀면 되겠다는 생각을 했고, 처음에 해당 방식으로 구현했다. 포인트는 queue.pop(0
```python
n, k = map(int, input().split(" "))

queue = [str(i) for i in range(1, n+1)]
result = []

while queue:
    for i in range(k-1):
        queue.append(queue.pop(0))
    result.append(queue.pop(0))

answer = ", ".join(result)
print('<{}>'.format(answer))
```
