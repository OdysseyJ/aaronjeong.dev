### 접근
1. stack에 넣어두고, 특정 인덱스에 있는 원소와 스택의 top을 계산해서 맞는 stack의 원소자리에 전부 정답을 처리해준다.
```python
n = int(input())
nums = [int(i) for i in input().split(" ")]

result = [-1] * n
stack = []

for i in range(n):
    while stack and nums[stack[-1]] < nums[i]:
        result[stack.pop()] = nums[i]
    stack.append(i)

print(*result)
```
