### 접근
1. 완전 이진 트리에서 in-order 순회 결과가 주어졌을 때 트리의 계층 구조를 표현하는 문제다.

in-order의 특징은 탐색의 가운데 결과값이 항상 루트가 되기 때문에

루트를 기준으로 좌/우 배열을 분리해주면 새로운 서브트리를 구할 수 있다.
```python
import sys
sys.setrecursionlimit(100000)
input = sys.stdin.readline


def solve(n, start, end):
    if n == k:
        return
    d = start + int((end-start)/2)
    root = in_order[d]
    results[n].append(root)
    solve(n+1, start, d-1)
    solve(n+1, d+1, end)


k = int(input())
in_order = input().split()
results = [[] for _ in range(k)]
solve(0, 0, 2**k-1-1)
for result in results:
    print(" ".join(result))
```
