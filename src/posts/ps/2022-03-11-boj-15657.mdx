### 접근
1. 전형적인 백트래킹 문제다. 백트래킹에 들어가기 위한 조건만 조절해주면 쉽다.

```python
import sys
sys.setrecursionlimit(100000)
input = sys.stdin.readline


def back_tracking(arr):
    if len(arr) == m:
        for i in arr:
            print(i, end=" ")
        print()
        return

    for num in nums:
        if not arr or arr[-1] <= num:
            arr.append(num)
            back_tracking(arr)
            arr.pop()


n, m = map(int, input().split())
nums = sorted([int(i) for i in input().split()])
back_tracking([])
```
