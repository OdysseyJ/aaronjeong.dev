#### 접근
1. 백트래킹으로 구하면 된다. 이때 포인트는 T를 S까지 줄여가면서 해야 시간초과가 안남.

줄여 나갈 때 재귀를 이용해서 뒤집은 문자열에서 B를 빼서 한번, 그리고 A를 끝에서 빼서 한번 트래킹 하면 된다.
```python
S = input()
T = input()
found = False


def solve(ss):
    global found
    if len(ss) < len(S):
        return
    if ss == S:
        found = True
        return

    if ss[0] == 'B':
        ss = ss[::-1]
        ss = ss[:-1]
        solve(ss)
        ss = ss + 'B'
        ss = ss[::-1]

    if ss[-1] == 'A':
        ss = ss[:-1]
        solve(ss)


solve(T)
print(1 if found else 0)
```
