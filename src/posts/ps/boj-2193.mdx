### 접근
1. dp[n][i] = i으로 끝나는 n자리 2친수
```python
n = int(input())

# dp[n][i] = i으로 끝나는 n자리 2친수
dp = [[0 for _ in range(3)] for _ in range(91)]

dp[1][0] = dp[2][1] = 0
dp[1][1] = dp[2][0] = 1

for i in range(3, 91):
    dp[i][0] = dp[i-1][0] + dp[i-1][1] # 0 앞에는 0, 1 둘다 가능
    dp[i][1] = dp[i-1][0] # 1 앞에는 0만 가능

print(sum(dp[n]))
```
