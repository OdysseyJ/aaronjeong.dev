```python
t = int(input())
for _ in range(t):
    n = int(input())
    univs = [input() for _ in range(n)]
    m = -1
    name = ""
    for univ in univs:
        s = univ.split(" ")[0]
        l = int(univ.split(" ")[1])
        if l > m:
            m = l
            name = s
    print(name)
```
