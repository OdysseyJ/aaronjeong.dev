### 접근
0. 먼저 d에서 n만큼 읽으면서 총 숫자가 몇번 나왔는지를 저장해둔다.
1. stack에 넣어두고, 특정 인덱스에 있는 원소와 스택의 top을 계산해서 맞는 stack의 원소자리에 전부 정답을 처리해준다.
```python
n = int(input())
nums = [int(i) for i in input().split(" ")]

d = {}
for num in nums:
    if d.get(num) is None:
        d[num] = 1
    else:
        d[num] += 1

result = [-1] * n
stack = []

for i in range(n):
    while stack and d[nums[stack[-1]]] < d[nums[i]]:
        result[stack.pop()] = nums[i]
    stack.append(i)

print(*result)
```
