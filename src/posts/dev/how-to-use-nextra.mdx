Nextra란, NextJS 를 기반으로 한 static site generator이다.

정적으로 블로그를 빌드하는 github pages나, gatsby같은 솔루션들이 많이 있지만, NextJS를 사용해 왔었기 때문에 익숙하기도 하고, 무엇보다 vercel에서 starter pack을 굉장히 쉽게 제공하기 때문에 손을  많이 거치지 않아도 되서 좋았다.

Gatsby도 리액트 기반의 static generator라서 굉장히 개발자들 사이에서 많이 사용되고 있는데, 마찬가지로 Nextra에서도 React component와 함께 사용이 가능하다.

자세한 내용은 Nextra 홈페이지에서 살펴보자.

https://nextra.vercel.app/

  return (
    <>

          rel="alternate"
          type="application/rss+xml"
          title="RSS"
          href="/feed.xml"
        />
        <link
          rel="preload"

          as="font"
          type="font/woff2"

        />
      </Head>
