### 재해 복구
- 재해 복구는 재해에 대해 대비하고, 복구하는 과정 전반을 말함
- 재해 복구의 종류
    - On-premise => On-premise : 전통적인 방식, 매우 비쌈
    - On-premise => AWS Cloud : 재해 발생 시 AWS 클라우드 이용
    - AWS Cloud Region A => AWS Cloud Region B : 리전간 복제
- 2가지 용어를 정의해야한다
    - RPO : Recovery Point Objective (복구 시점 목표)
    - RTO : Recovery Time Objective (복구 시간 목표)
