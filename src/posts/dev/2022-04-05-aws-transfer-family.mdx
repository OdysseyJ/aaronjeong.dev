### Transfer Family
- S3 or EFS로 파일을 전송할때 FTP 프로토콜을 사용하고 싶으면 사용
- 서포트되는 프로토콜
    - FTP (File Transfer Protocol)
    - FTPS (File Transfer Protocol over SSL)
    - SFTP (Secure File Transfer Protocol)
- 관리형, 확장가능, 신뢰할수 있고, 높은 가용성 (multi-AZ)
- 프로비전된 엔드포인트 + 데이터 전송 GB당 과금
- 유저의 인증 저장과 관리가 가능하다.
- 기존 인증 시스템과 통합 가능 (cognito, Microsoft Active Directory...)
