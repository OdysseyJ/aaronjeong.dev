### Storage Comparison
- S3 : Object Storage
- Glacier : Object Archival
- EFS : 리눅스 인스턴스를 위한 네트워크 파일 시스템, POSIX 파일시스템 사용
- FSx for Windows : 윈도우 서버를 위한 네트워크 파일 시스템
- FSx for Lustre : 높은 성능의 컴퓨팅 리눅스 파일 시스템
- EBS volumes : EC2 인스턴스를 위한 네트워크 스토리지
- Instance Storage : EC2 인스턴스를 위한 물리적 스토리지 (high IOPS)
- Storage Gateway : 파일 게이트웨이, 볼륨 게이트웨이(cached & stored), 테이프 게이트웨이
- Snowball / Snowmobile : 큰 데이터를 클라우드에 옮기기 위한 물리적 데이터베이스
- Database : 특정한 워크로드에서 사용 (인덱싱, 쿼리)
