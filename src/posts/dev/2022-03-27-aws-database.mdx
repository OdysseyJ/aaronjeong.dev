#### Database Types
- RDBMS(=SQL/OLTP): `RDS`, `Aurora` = join과 정규화된 데이터
- NoSQL database: `DynamoDB`(~JSON), `ElasticCache`(key/value), `Neptune`(graphs) - no join, no SQL
- Object Store `S3`(for big) / `Glacier`(for backups/archives)
- Data Warehouse(=SQL Analytics/BI): `Redshift`(OLAP), `Athena`
- Search: `ElasticSearch`(JSON) - free text, unstructured searches
- Graphs: `Neptune` - display relationships between data
